Welcome to the github project of orthologr.

There exists a vast diversity of published and implemented gene orthology detection methods and techniques. This R package is an attempt to use the strength and reproducibility of the R package philosophy and aims to combine common open source software that has been developed for gene orthology detection to provide a orthology detection pipeline for the R framework.

You can download the current version of the package as following in R:

library(devtools)
install_github('orthologr')
